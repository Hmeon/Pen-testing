- CVSS v3.1 Score: 9.8
- CVSS v4: 미측정
- CWE: CWE-912
- OWASP Top10:2025 (추정 매핑): A05 Security Misconfiguration
- 상태: Open

### 요약

### 영향(C/I/A)
기밀성/무결성 영향이 높고, 가용성 영향도 연쇄적으로 발생할 수 있는 고위험 항목으로 분류한다.

### 재현 조건
승인된 이중 VM 실습 환경에서 취약 서비스 노출 상태가 유지되어야 한다.

### 증거
- 근거 ID: `PCAP-001`, `LOG-001`, `IMG-001`, `IMG-002`
- 대표 파일: `img/fig_7-1.png`, `img/fig_7-2.png`

### 개선 권고
취약 서비스 제거, 공식 배포 해시 검증 의무화, 서비스 접근 원천 제한 정책을 우선 적용한다.

### 재검증 기준
취약 경로 세션 미성립 + 관련 포트 비정상 노출 0건.

### 요약

### 재검증 기준
